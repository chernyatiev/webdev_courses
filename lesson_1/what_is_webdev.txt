Frontend:
   html
   css
   javascript (CoffeScript, ClojureScript)


Backend:
   Web-Server(Nginx, LightHttp)
   Application(Python, PHP, JavaScript, Ruby)
   DataBase(MySQL, Postgre, Oracle, MongoDB,...)
   Cache (Redis,Memcached)

============

Запускаем простой веб-сервер. В командной строке наберите команду:
$ python -m SimpleHTTPServer

По умолчанию сервер работает на порту 8000, он отдает статические файлы из папки, где он запущен.
Посмотреть работу сервера можно, набрав в браузере:
localhost:8000
